# サイバー攻撃 その瞬間 社長の決定

- **著者**: 関通サイバー攻撃対策室
- **企業**: 株式会社関通（東証グロース上場、総合物流企業）
- **事案**: 2024年9月12日に発生したランサムウェア攻撃の実録

## 本書について

東証上場の物流企業・関通が、ランサムウェア攻撃によりサーバー全暗号化・500社以上の取引先に影響という未曾有の危機に直面し、約1ヶ月で復旧するまでの実話。経営者の視点で日々の意思決定・葛藤・教訓を赤裸々に語る。
