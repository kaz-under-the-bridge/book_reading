# 第3章 Note

## 全体所感

「業績評価」の章。評価プロセスにおける**情報の非対称性**を理解し、その「ショック緩和」を図るための知識を提供している。評価する側・される側双方に有用。

## 情報の非対称性と業績評価

業績評価において、評価者（マネージャー）と被評価者（IC）の間には認識のギャップが存在する。

- 自分が成果を出していると思っていても、マネージャーがそれを認識していない可能性
- 逆に、自分が気づいていない評価ポイントをマネージャーは見ている可能性
- 評価結果を受けた際の感情的な反応（ショック）を緩和するために、このギャップを事前に理解しておく

## フィードバック能力のばらつき

> マネージャーの誰もが等しくフィードバック能力を持っているわけではないことを認識しておくべきです。

これは重要な認識。

- 優れたフィードバックを受けられるかどうかは、マネージャー次第
- フィードバックの質が低い場合、自ら補完する必要がある（同僚からのフィードバック、自己評価の精度向上など）
- マネージャーとして読む場合: 自分のフィードバック能力を客観視し、改善の余地を探る

## 評価側としての視点

リーダー/マネージャーとして評価する側で読むと：

- 被評価者が持つ不安やギャップを理解する
- 情報の非対称性を解消するためのコミュニケーション（期待値のすり合わせ、進捗確認）
- フィードバックの質を高める意識
