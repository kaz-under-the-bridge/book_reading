# 第6章 Note

## 全体所感

「転職」の章。技術面接の準備、レベル調整（昇格/降格）、オンボーディングなどを扱っている。

海外（主に米国）基準で日本企業と異なる部分が多いため、**軽く読み流すぐらいで良い**。

## 日本との差異

### 技術面接
- 日本では「ホワイトボードコーディング」「システムデザイン面接」は一般的ではない
- LeetCode的な対策が必須な企業は限定的（外資系、一部スタートアップ）
- 日本企業では経歴・実績ベースの面接が主流

### レベル調整
- 転職時のレベル昇格/降格の交渉は、日本では一般的ではない
- 海外のように「レベルを下げてでも入社し、後から昇進する」戦略は文化的に取りにくい

### オンボーディング
- 海外の「90日プラン」などの体系的なオンボーディングは参考になる
- ただし、日本企業では形式化されていないことが多い

## 参考になる点

- 新しい環境に入った際の学習戦略（技術スタック、チーム文化、ビジネスコンテキスト）
- 「最初の90日」で信頼を築く意識
- オンボーディング期間中の期待値調整

これらは日本企業でも普遍的に適用できる。
